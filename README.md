# Smart To-Do Manager

A feature-rich CLI Task Manager built with Python.

## Features
- Add, Edit, Delete tasks
- Priority system (High / Medium / Low)
- JSON file persistence (auto save/load)
- Search and Filter tasks
- Statistics dashboard with progress bar
- Exception handling — crash proof

## How to Run
python todo.py

## Tech Stack
- Python 3
- OOP (Classes)
- JSON (File Storage)
- Standard Library only
